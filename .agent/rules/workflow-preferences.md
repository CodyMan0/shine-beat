# Workflow Preferences

### Default Flow
1. **Planning First** – For major work, plan and await approval.
2. **Iterate in Steps** – Break down tasks into small, testable changes.
3. **Build Verification** – Run `npm run build` after changes to verify TypeScript + Vite compilation.
4. **Approval Required** – For major changes, pause and wait for confirmation.

### Commands
| Command | Purpose |
|---------|---------|
| `npm run dev` | Start dev server |
| `npm run build` | TypeScript check + Vite build |
| `npm run preview` | Preview production build |
